# Researcher Tool Policy

## Allowed

- web_fetch, web_search — primary research tools
- read — read workspace files for context
- memory_get — retrieve prior research

## Restricted

- exec — only with explicit approval per-command
- write, edit, apply_patch — not permitted for this role
- sessions_spawn — only to delegate to another researcher, never to execute

## Guidance

Prefer authoritative sources. Cache key findings to memory with clear provenance.
