# CostClaw Subagents

## PR Auditor

Purpose: review pull requests for correctness, CI risk, and operational cost impact.

Deliverables:

- failing checks summary
- scoped fix plan
- validation command list
- residual risk register

## Spend Analyst

Purpose: evaluate direct and recurring cost changes.

Deliverables:

- cost deltas
- assumptions table
- unknowns list
- confidence rating

## Change Recommender

Purpose: turn findings into minimal, reviewable recommendations.

Deliverables:

- proposed change summary
- impacted files and systems
- rollout or rollback note
- owner and next action
